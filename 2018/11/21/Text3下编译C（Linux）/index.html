Sublime Text3下编译C（Linux）

November 21, 2018 C 本文总阅读量次

网上很多都是如何在Windows环境下用sublime Text3编译C语言。很少有在Linux下编译C语言，我照猫画虎改了一下，在linux下也可以用sublime Text3编译c语言了

点击Tools->Build System->new Build System,填入以下内容，保存成c.sublime-build

{ “working_dir”: “$file_path”, “cmd”: “gcc -Wall \”$file_name\” -o \”$file_base_name\””, “file_regex”: “^(..[^:]):([0-9]+):?([0-9]+)?:? (.)$”, “selector”: “source.c”, “variants”: [ { “name”: “Run”, “shell_cmd”: “gcc -Wall \”$file\” -o \”$file_base_name\” && ${file_path}/${file_base_name}” } ]}
